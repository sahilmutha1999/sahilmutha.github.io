My Portfolio Some Of My Works

BERT LLM based Optimized Financial Market Sentimental & Portfolio Analysis

Problem:- Investors have a problem in identifying right companies to invest and to analyze potential downturns of a company before investing.

Solution:- In this project, I utilized advanced language models like BERT to analyze market sentiment from news articles. Using natural language processing techniques, I quantified sentiment for informed decision-making in risk assessment and portfolio rebalancing. Additionally, I employed statistical methods and simulation to forecast portfolio performance, enhancing preparedness for market fluctuations.

Passion Hub - Fullstack application

Problem:- Lack of platform to connect with like-minded individuals sharing same passion as yours and network with them and share knowledge.

Solution:- PassionHub provides an easy-to-use web platform for individuals to connect with others who share their interests, offering personalized recommendations, group discussions, and direct messaging. It employs React and TypeScript for client-side rendering, Redux for state management, and Node.js with Express.js for backend development. MongoDB serves as the database, while RESTful APIs are created for seamless interactions. Postman is utilized for API testing, ensuring secure transactions and efficient content discovery for users.

Autonomous agricultural robot using R-CNN

Problem:- Tomato farms are spread in large acres of land and farmers invest lot of time in grinding, classifying and plucking right tomatoes from farms.

Solution:- Designed a functional agricultural robot to classify and pluck ripened tomatoes. Developed an algorithm to classify raw and ripened fruits using DarkNet-53 architecture for feature extraction and R-CNN for proposing ROI within feature maps. Performed data cleaning using K-means clustering to group similar images and perform statistical methods for identifying outliers

Java based Person Identity Management System

Problem:- Existing identity verification methods face inefficiencies and security vulnerabilities, hindering seamless collaboration among universities, certification authorities, research publications, and employers.

Solution:- The Personal Identity Management System leverages object-oriented programming principles and harnesses the power of Java's extensive libraries and frameworks, such as JDBC for seamless database connectivity. By integrating advanced technologies like JSON data and robust verification management systems, it redefines identity verification processes. This platform facilitates secure and efficient communication among universities, certification authorities, research publications, and employers. Through Java's versatility, scalability, and reliability, it establishes a new benchmark for identity and achievement verification in the digital age.

AI based Genomic Automation Accelerator: Leveraging TensorFlow for Drug Discovery

Problem:- Manual analysis of genomic data hampers drug discovery efficiency and precision medicine efforts.

Solution:- Employed Convolutional Neural Networks (CNNs) within an AI-powered drug discovery platform to automate genomic data analysis. Leveraging Python as the primary language and TensorFlow with Keras for implementation, I accelerated drug development by swiftly identifying potential candidates through efficient pattern recognition in genomic data.
